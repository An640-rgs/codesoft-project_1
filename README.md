# codesoft-project_1
 PROJECT 1 ROCK PAPER SCISSOR GAME :- A simple command-line Rock Paper Scissors game built with Python. This project is a beginner-friendly implementation of the classic game, allowing a user to play against the computer.
  
PROJECT 2 PASSWORD GENERATOR :-A simple yet powerful password generator built with Python. This project creates strong, random passwords using a mix of uppercase and lowercase letters, numbers, and special characters to enhance security.


PROJECT 3 CALCULATOR:-A basic calculator application built with Python that performs standard arithmetic operations. This project is ideal for beginners to understand fundamental Python concepts and simple logic implementation.
